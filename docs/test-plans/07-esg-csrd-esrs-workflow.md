# ESG Testplan — CSRD / ESRS / EU-Taxonomie / GHG Protocol

## Normenkonformer End-to-End Workflow

### Phase 1: Wesentlichkeitsanalyse (ESRS 1 / CSRD Art. 19a)

- [ ] G1.1: ESG-Dashboard aufrufen (/esg)
- [ ] G1.2: Doppelte Wesentlichkeitsanalyse starten (/esg/materiality)
- [ ] G1.3: Impact-Materiality bewerten (Inside-Out)
- [ ] G1.4: Financial-Materiality bewerten (Outside-In)
- [ ] G1.5: Wesentliche Themen identifizieren

### Phase 2: Datenpunkte & Kennzahlen (ESRS)

- [ ] G2.1: ESRS-Datenpunkte zuweisen (/esg/datapoints)
- [ ] G2.2: Verantwortliche für Datenerhebung zuweisen
- [ ] G2.3: ESG-Kennzahlen erfassen (/esg/metrics)
- [ ] G2.4: Datenqualität prüfen (gemessen/geschätzt/berechnet)

### Phase 3: THG-Emissionen (GHG Protocol)

- [ ] G3.1: Scope 1 Emissionen erfassen (/esg/emissions)
- [ ] G3.2: Scope 2 Emissionen erfassen (standort-/marktbasiert)
- [ ] G3.3: Scope 3 Kategorien erfassen (/esg/emissions/scope3)
- [ ] G3.4: Lieferketten-Emissionen (/esg/emissions/supply-chain)
- [ ] G3.5: Emissionsfaktoren zuordnen

### Phase 4: Ziele & Maßnahmen

- [ ] G4.1: Klimaziele definieren (/esg/targets)
- [ ] G4.2: Reduktionspfad planen (SBTi-kompatibel)
- [ ] G4.3: Maßnahmen verknüpfen

### Phase 5: EU-Taxonomie

- [ ] G5.1: Wirtschaftsaktivitäten bewerten (/esg/taxonomy)
- [ ] G5.2: 6 Umweltziele prüfen (Klimaschutz, Anpassung, Wasser, Kreislauf, Verschmutzung, Biodiversität)
- [ ] G5.3: Alignment-Quote berechnen (Umsatz/CapEx/OpEx)

### Phase 6: Berichterstattung

- [ ] G6.1: ESG-Bericht generieren (/esg/report)
- [ ] G6.2: ESRS-Compliance prüfen
- [ ] G6.3: Tax CMS prüfen (/tax-cms)
